# Test Images

Place test images in this directory to test the age and gender prediction system.

## Recommended Test Images

For best results, use images with:
- Clear, frontal faces
- Good lighting
- Various age groups (children, young adults, middle-aged, elderly)
- Both male and female subjects
- Single or multiple faces

## Supported Formats

- JPEG (.jpg, .jpeg)
- PNG (.png)
- BMP (.bmp)
- TIFF (.tiff)
- WebP (.webp)

## Testing Tips

1. Start with high-quality images with clear faces
2. Test with different age groups to verify accuracy
3. Try images with multiple faces
4. Test edge cases (side profiles, poor lighting, etc.)

You can find free test images from:
- [Unsplash](https://unsplash.com/)
- [Pexels](https://www.pexels.com/)
- [Pixabay](https://pixabay.com/)

Make sure to use images you have the right to use for testing purposes.
